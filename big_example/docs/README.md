# Big Example Service

This directory contains a large example of a Python service with a three-tiered architecture.

## Structure

- `domine`: Contains the domain models for the service.
- `repo`: Contains the repository classes for accessing the domain models.
- `service`: Contains the service classes that implement the business logic.
- `data`: Contains sample data files.
- `docs`: Contains documentation.
- `complex_logic`: Contains files with complex business logic and many imports.
- `main.py`: The entry point for the application.
- `config.py`: Configuration settings.
- `utils.py`: Utility functions.
